# Mini Web Projects

16 bite-sized browser games and tools, all built with HTML, SCSS, and vanilla JavaScript.  
No frameworks, no sign-ups, no installs. Just click and play.  
Built for learning, tinkering, and having a bit of fun.

## All Projects

| Project | Description |
|---------|-------------|
| [Breakout](projects/Breakout/) | A classic Breakout game. Bounce the ball, break the bricks, beat the level. |
| [Calculator](projects/Calculator/) | A functional calculator with basic arithmetic and a hidden surprise. |
| [Connect Four](projects/Connect%20Four/) | Drop discs and line up four in a row vertically, horizontally, or diagonally. |
| [Dice Roller](projects/Dice%20Roller/) | Roll up to 36 dice with animations and see the total sum. |
| [Finspan Scoreboard](projects/Finspan%20Scoreboard/) | A convenient scoreboard for tracking points in Finspan. |
| [Frogger](projects/Frogger/) | Guide your frog across roads and rivers. Avoid cars and ride the logs. |
| [Memory](projects/Memory/) | Find matching pairs of cards in this classic memory game. |
| [Rock Paper Scissors](projects/Rock%20Paper%20Scissors/) | Challenge the computer to a duel of wits. Best of luck. |
| [Space Invaders](projects/Space%20Invaders/) | Defend Earth from waves of aliens with auto-fire and enemy attacks. |
| [Stopwatch](projects/Stopwatch/) | A clean stopwatch with start, stop, and reset. |
| [Tarot Reader](projects/Tarot%20Reader/) | Draw your cards and reveal your fate with saved reading history. |
| [Tic Tac Toe](projects/Tic%20Tac%20Toe/) | The timeless 3×3 grid game. Face off against a friend. |
| [Time Calculator](projects/Time%20Calculator/) | Add or subtract hours, minutes, and seconds with ease. |
| [Truth or Dare](projects/Truth%20or%20Dare/) | Classic and Couples modes. No skips, just fun. |
| [Wellness Calculator](projects/Wellness%20Calculator/) | A full wellness dashboard with BMI, BMR, TDEE, ideal weight, and more. |
| [Whack a Mole](projects/Whack%20a%20Mole/) | Smack those moles as fast as you can before time runs out. |

## Features

- Interactive and user-friendly interfaces
- Responsive designs for desktop, tablet, and mobile
- No installations, sign-ups, or frameworks required
- Each project is self-contained in its own folder
- Search and filter on the homepage to quickly find projects

## Installation

### Just play
1. Clone the repo:  
   `git clone https://github.com/Gloomcaller/Mini-Web-Projects.git`
2. Open the root `index.html` to browse all projects, or dive into any folder and open its `index.html`.

### Tinker with the code
- **Live Server** – preview changes instantly ([VS Code extension](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer))
- **Live Sass Compiler** – watch and compile SCSS to CSS ([VS Code extension](https://marketplace.visualstudio.com/items?itemName=glenn2223.live-sass))

## Technologies Used

- **HTML** – Structure and layout
- **CSS / SCSS** – Styling, animations, responsive design
- **JavaScript** – All logic, interactivity, and game mechanics
- **JSON** – Project data and configuration
